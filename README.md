# Mini-App для Kayo Bot

Это Mini-App для заполнения анкеты фурсоны в Telegram-боте Kayo.

## 🐾 Функции

- Заполнение анкеты фурсоны
- Загрузка фото референса
- Сохранение в базу данных бота
- Интеграция с Telegram Web App API

## 📦 Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер:
```bash
python webapp/server.py
```

3. Откройте в браузере: `http://localhost:8080`

## 🐾 Интеграция с ботом

1. Добавьте обработчики в `main.py`:
```python
from bot.handlers.webapp import register_webapp_handlers
register_webapp_handlers(application)
```

2. Настройте URL Web App в Telegram BotFather:
```
/setwebapp
```

## 🐾 Поля анкеты

- **Имя фурсоны** (обязательное)
- **Вид** (обязательное)
- **Дата рождения**
- **Возраст**
- **Ориентация**
- **Город**
- **Фото референса**
- **Ищет**
- **Тип личности**

## 🐾 Технические детали

### Сервер
- Использует встроенный HTTP-сервер Python
- Обрабатывает POST запросы на `/save_profile`
- Сохраняет данные в SQLite базу данных

### Клиент
- HTML/CSS/JavaScript интерфейс
- Интеграция с Telegram Web App API
- Загрузка фото через FileReader API

### База данных
- Таблица `user_profiles` в базе бота
- Хранит все поля анкеты
- Обновляется при изменении

## 🐾 Запуск в продакшене

1. Разверните сервер на хостинге (Heroku, VPS, etc.)
2. Настройте HTTPS (обязательно для Telegram Web App)
3. Укажите URL в настройках бота через BotFather
4. Добавьте кнопку в групповом чате

## 🐾 Пример использования

1. Пользователь нажимает кнопку "Заполнить анкету"
2. Открывается Mini-App с формой
3. Пользователь заполняет поля
4. Нажимает "Сохранить"
5. Данные отправляются на сервер
6. Сервер сохраняет в базу данных бота
7. Mini-App закрывается

## 🐾 Безопасность

- Проверка user_id через Telegram Web App API
- Валидация данных на сервере
- Защита от SQL-инъекций
- HTTPS для передачи данных
